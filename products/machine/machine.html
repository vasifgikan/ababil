ОБОРУДОВАНИЕ ДЛЯ ПРОИЗВОДСТВА СИГАРЕТ

ОБОРУДОВАНИЕ ДЛЯ ПРОИЗВОДСТВА СИГАРЕТ С ПРИМЕНЕНИЕМ ИНДИВИДУАЛЬНЫХ ПРОИЗВОДСТВЕННЫХ РЕШЕНИЙ

На протяжении многих лет «Baltic Tobacco Group» поставляет оборудование для производства сигарет с применением индивидуальных производственных решений для производства сигарет, в том числе сигаретоделательные машины, сигаретоупаковочные машины, машины для изготовления фильтров, для упаковывания в пленку, для штамповки, а также укладочные машины. Наша цель — поставлять качественное оборудование, и за прошедшие десятилетия мы достигли уверенного прогресса в деятельности по поставкам оборудования. Мы гордимся тем, что поставляем 90% наименований машинных деталей.

  • Сигаретоделательная машина
  • Машина для изготовления полых трубок для сигарет

  • Машина для упаковки в алюминиевую фольгу
  • Машина для упаковки пачек
  • Машина для упаковки в картонные коробки
  • Сигаретоделательная машина MK8
  • Сигаретоупаковочная машина HLP2
  • Машина для обратной транспортировки бракованных сигарет
  • Машина для изготовления двойных фильтров
  • Машина для изготовления фильтров (низкая скорость)
  • Машина для изготовления фильтров (высокая скорость)
  • Машина для штампования акцизных марок на пачки с откидной крышкой (HLP)
  • Машина для полной упаковки в прозрачную пленку
  • Машина для покрытия прозрачной пленкой
  • Машина для упаковки сигарет в пачку с откидной крышкой
